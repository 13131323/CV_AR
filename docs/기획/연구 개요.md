# 연구 설명

장면 이해 결과를 아바타 행동으로 변환

Systemic paper

기능

- 공간 이해 : 공간 파악, 사물 용도 파악
- 행동 결정 : LLM 바탕 사고로 행동 결정
- 동작 생성 : 아바타 행동

공간 이해

- 사물의 이해
- 물리적 수치
- 어포던스 파악 (행동 가능성)

행동 결정

- 상황 이해
- 장면 정보를 LLM에게 전달 -> 상황에 맞는 행동 레이블 결정

행동 표현 생성

- 이동 가능 영역 생성 (네비게이션 메시)
- 장애물 정의 (정적 장애물 + 동적 장애물 구분)
- 경로 계획
- 행동 검증 포함

동작 생성

- 결정된 행동을 현실 지형에 맞춰 물리적으로 어색하지 않게 렌더링 (물리 일관성)

공간 이해 기술

- 3D Bounding Box
- 수평면 데이터 추출
- depth (거리) 추출
- affordance 후보 추출

(v) 공간 이해 -> 행동 결정 -> 행동 표현 생성 -> 동작 생성 : 뭘 할지 먼저 결정하고, 그 다음 어떻게 할지 계획
(x) 공간 이해 -> 행동 표현 생성 -> 행동 결정 -> 동작 생성 : 공간 가능성을 먼저 계산하고, 그 중에서 결정

1. 이 과정에서 컴퓨터 비전 연구 요소는?

장면 이해 결과를 행동 가능한 표현으로 재구성하는 것이 핵심 CV 연구 요소

- 장면 이해 결과를 행동에 맞게 재구성하는 과정

시맨틱 세그멘테이션 및 어포던스 추출

- 객체 분류
- 이미지 내 픽셀들이 가지는 물리적 의미 분석

3D 재구성 및 공간 기하학 분석

- SLAM과 3D Vision
- 카메라 영상에서 의자, 테이블, 사람 등을 실시간으로 탐지
- depth 정보와 결햅해서 각 물체의 3D 위치(x, y, z 좌표)와 크기를 추정
- 2D 카메라 이미지를 3D 공간 데이터로 변환 후 현실 세계의 좌표계와 일치

깊이 추정 (Depth Estimation)

- RGB-D 카메라 사용하면 depth가 직접 나옴
- 일반 카메라만 쓰면 MiDaS나 Depth Anything 같은 단안 깊이 추정 모델을 사용해야 됨
- 모델이 각 픽셀까지의 거리 추정

평면 감지 (Plane Detection)

- ARCore, PlaneRCNN
- 수평면(바닥, 테이블 위)과 수직면(벽) 감지
- 법선 벡터를 추출해서 어느 면에 서 있어야 하는지, 어디가 이동 가능한 지형인지 판단

시맨틱 세그멘테이션 (Semantic Segmentation)

- 픽셀 단위로 바닥, 벽, 의자 등을 구분
- 단순 객체 탐지보다 더 정밀한 공간 이해 가능
- NavMesh 생성할 때 이동 가능/불가능 영역을 나누는 데 직접 활용

물체 포즈 추정 (Pose Estimation)

- MediaPipe, OpenPose
- 사물의 6자유도 포즈 추정 (6-DoF)
- 사물의 앞면, 뒷면, 높이를 비전 데이터만으로 정확히 계산

조명 및 쉐이딩 일관성

- 주변 광원 정보를 비전 데이터로 추정 (Inverse Rendering)
- 아바타의 그림자와 밝기를 조절

1. 동작 생성을 할 수 있는가?

MDM (Motion Diffusion Model)

- 텍스트나 행동 레이블을 입력받아 SMPL 형식의 3D 인체 모형 생성
- 골격 애니메이션을 생성해줌
- GitHub에 공개돼 있음

IK 활용 (Inverse Kinematics)

- 발바닥이나 엉덩이 같은 끝단(End-effector)의 위치만 현실 지형에 고정하면, 시스템이 물리 법칙에 맞게 나머지 관절(무릎, 허리 등)의 각도를 자동으로 계산
- 유니티나 언리얼 엔진 같은 툴에서 지원

모션 워핑 (Motion Warping)

- 미리 만들어진 기본 동작(걷기, 앉기)을 현실의 타겟 좌표에 맞춰 실시간으로 비틀거나 늘리는 기술

- YOLO는 ScanNet 데이터셋으로 파인튜닝 가능

현실 지형 조건을 추가로 넣기 위한 경량 후처리 필요 (post processing)

- 학습이 아닌 규칙 기반 보정

1. AI 융합합응용에 포함되는가?

AI 융합응용 : 단일 AI 기술이 아니라 여러 AI 기술을 실제 응용 문제에 통합하는 연구

AI(LLM) + Computer Vision + XR(Extended Reality)

컴퓨터 비전
