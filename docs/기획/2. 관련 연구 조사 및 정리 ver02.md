# SAM

1. 한 줄 정리

프롬프트 입력을 기반으로 사전 학습 없이 이미지 내의 모든 객체를 분할(Segmentation)하는 시각 파운데이션 모델

- 점(Point), 박스(Bounding Box), 텍스트(Text) 입력 → 이미지의 객체 분할 인식

어떤 이미지에서든 프롬프트 기반으로 모든 객체를 정교하게 분할할 수 있는 범용 비전 기초 모델

범용 이미지에서 작동하는 세그멘테이션 파운데이션 모델

1. 기존 문제점

기존 세그멘테이션 모델은 특정 데이터셋에 과적합되어 있어서
→ 처음 보는 환경이나 새로운 물체에서 인식 성능이 급격히 저하됨 (Zero-shot)

- 이로 인한 데이터셋 의존성이 심했음

특정 클래스만 인식하는 한계가 있었음

세그멘테이션 작업을 위한 조건이 복잡했음

1. 논문 아이디어

대규모 언어의 프롬프팅 개념을 컴퓨터 비전에 도입하여
→ 대규모 데이터셋 학습을 통해 제로 샷 일반화 능력을 갖춘 파운데이션 모델 구축

- 데이터셋으로 SA-1B 사용

대규모 데이터로 학습하면 프롬프팅만으로 모든 객체 분할이 가능하다

1. 핵심 아키텍처

SAM = Image Encoder + Prompt Encoder + Mask Decoder

Image Encoder

- 대용량 비전 트랜스포머 기반(ViT)으로 이미지 전체의 고차원 특징 추출
- 이미지 전체의 의미론적 이해
- 1번만 연산

Prompt Encoder

- 점, 상자, 텍스트 등의 프롬프트 입력을 위치 임베딩 벡터로 변환
- “이 위치의 객체를 분리해라” 같은 역할

Mask Decoder

- 이미지 특징과 프롬프트 임베딩을 상호 참조(Cross-attention)하여 최종 픽셀 마스크 레이어를 실시간으로 생성
- 이미지 인코더와 프롬프트 인코더로 변환한 두 임베딩을 결합하여 → 마스크 생성
- 출력 마스크는 고해상도 세그멘테이션
    - 객체 영역 추출

1. 핵심 파이프라인

Image Input → Vit 백본 임베딩 → 사용자 프롬프트 주입 → Cross-Attention 디코딩 → 최종 다중 마스크(Masks) 출력

1. 핵심 모듈

Image Encoder (Heavy)

- 입력 이미지 → 패치 임베딩 → 비전 트랜스포머 → 고해상도 특징 맵
- 이미지 특징 추출
- 입력 이미지의 전반적인 시각적 맥락과 특징 패턴을 파악하여
→ 16배 축소된 기능 맵(Feature Map)으로 변환
- 고성능 연산이 필요하므로 원본 이미지마당 한 번만 수행

Prompt Encoder (Flexible)

- 프롬프트 입력 → 프롬프트 임베딩
- 사용자 입력 해석
- 다양한 형태의 사용자 프롬프트 처리
- 사용자가 찍은 점, 사각형, 텍스트 토큰을 → 고정된 크기의 밀집/희소 임베딩 벡터 공간으로 실시간 매핑

Mask Decoder (Lightweight)

- 이미지 인코더 특징, 프롬프트 인코더 결과 입력 → 트랜스포머 디코더 → 업 샘플링 → 시그모이드 활성화 → 다중 마스크 생성
- 세그멘테이션 생성
- 이미지 임베딩과 프롬프트 임베딩을 양방향으로 Attention 처리하여 모호성 제거
- 물체의 최종 테두리 경계를 정확한 픽셀 단위 마스크 정보로 매핑하여 출력

SA-1B Dataset

- 범용성 확보를 위한 대규모 데이터셋

Loss Function 및 학습

- Focal Loss → Dice Loss → 총 Loss 계산
    - Focal Loss에서 불균형 데이터 처리
    - Dice Loss는 마스크 경계 정확도
- 모든 모듈을 함께 최적화

1. 실험 결과

제로샷 환경에서 SOTA 모델보다 뛰어난 엣지 검출 및 분할 성능 달성

- 대규모 데이터 구축 덕분
    - 대규모 데이터를 통한 일반화
- 대규모 자율 루프 학습 체계를 구축했기 때문
    - 이 과정에서 사람이 직접 라벨링한 데이터, AI 보조 데이터, 완전 자동 생성 데이터로 구성된 1100만 장의 이미지와 11억 개의 마스크 데이터셋(SA-1B) 기반의 대규모 자율 루프 학습 체계를 구축했기 때문
- 프롬프트 제어
    - 기존에는 특정 객체를 학습한 모델을 사용했다면,
    SAM은 객체를 인식 후 프롬프트로 제어
    - 특정 클래스가 아닌 ‘무엇을 분리할지’를 입력받기 때문에 범용성이 커짐

1. 논문 의의

객체 분할 작업을 별도의 추가 학습 없이 즉석에서 수행할 수 있는 파운데이션 모델의 표준을 만듬

- 기존이 객체 위주의 세그멘테이션이었다면,
SAM은 파운데이션 모델을 만듬

세그멘테이션을 프롬프트 문제로 재정의

1. 한계점

텍스트 프롬프트 인지 능력이 상대적으로 약함

픽셀 단위 경계가 모호한 반사체, 투명체에서 경계 인식이 어려움

아주 작은 객체에서 미세한 디테일 포착 실패

객체가 겹칠 때 분리가 어려움

자연어 이해도가 재한적

동영상 스트림에서의 연속적인 추적(Tracking)에 취약함

- 프레임마다 일관성이 없음
- 시간적 정보가 무시됨

세그멘테이션만 수행하고 의미는 이해하지 못함

- 파운데이션 모델이기 때문에 따라오는 자연스러운 요소
- 어포던스 역시 마찬가지

모방리 실시간 처리 어려움

- ViT 기반이기 때문

시간 일관성 부족

- 영상에서 프레임마다 세그멘테이션이 흔들릴 수 있음

1. 결론

프롬프트 기반 세그멘테이션 파이프라인은 다양한 도메인의 비전 전처리 표준으로 자리 잡음

범용적이고 실시간으로 작동하는 기초 모델 제시

1. 내 연구에서는?

입력된 영상 데이터에서 아바타가 상호작용할 타겟 사물의 2D 영역 외곽선을 따내는 전처리 모듈

객체 분리

어포던스 추론 입력 생성

활용 파이프라인은 “RGB Frame → SAM → 세그멘테이션”

1. 궁금점

사용가능한 모델이 배포되어 있는가?

- 깃 허브가 있는가? 코드를 볼 수 있는가?
- 파인튜닝도 가능한가?

모델을 사용할 때 작동 시간, 필요한 하드웨어 자원은 어떤가?

- 저사양에서 실시간으로 작동시킬 수 있는가?

모델을 사용할 때 입력/출력 데이터는 어떻게 되는가?

내가 프롬프트도 입력해주어야 하는가?

- 내 연구에서는 “영상 입력 → 세그멘테이션 자동 분할”이 이루어져야 함

# Depth Anything V2

1. 한 줄 정리

전처리 기법 개선과 합성 데이터 학습을 통해 왜곡 없이 정교한 미터 단위의 깊이(Metric Depth)를 실시간으로 추정하는 범용 단안 깊이 추정 기초 모델

합성 데이터 기반 자기지도 학습으로 단일 이미지에서 실시간으로 고정밀 깊이를 추정하는 범용 깊이 추정 모델

단안 RGB 영상만으로 현실 공간의 깊이 Depth를 범용적으로 추정할 수 있는 파운데이션 Depth 모델

1. 기존 문제점

기존 단안 깊이 추정(Monocular Depth Estimation) 모델들은 유리창, 투명체, 얇은 전선 등의 경계를 뭉개버렸음

절대적인 거리가 아닌 상대적 깊이(Relative Depth)만 제공하여 정밀한 수치로 활용하기 어려웠음

기존 모델들은 특정 데이터셋에만 과적합되어 있었음

얇은 구조에 대해 세부 디테일이 손실됐음

LiDAR, Stereo Camera, Depth Camera 등의 Depth Sensor 의존도가 높았음

- 특수 하드웨어가 필요했다는 것

정확도와 속도의 트레이드 오프가 있었음

- 실시간 서비스 적용이 어려웠음

1. 논문 아이디어

정답 라벨이 완벽한 렌더린됭 가상 합성 데이터(Synthetic Data)로 교사 모델을 학습시킨 뒤,
→ 이를 통해 실제 이미지에 정밀한 의사 라벨을 부여(Pseudo-label)하여
→ 학생 모델을 대규모 학습시키자

- 증류 학습 or 자기 지도 학습 이용
    - Teacher가 데이터 간의 특징을 학습해서 액기스를 전수하는 것이 아닌,
    Teacher가 가상 데이터를 바탕으로 완벽한 데이터 기반 학습을 한 후,
    Teacher가 기존 데이터셋에 정밀한 라벨을 부여
    - 즉, 고품질 가상 합성 데이터를 만들기 위해 Teacher를 사용하고
    Student는 고품질 가상 합성 데이터로 학습하여 추론

즉, 완벽한 라벨이 있는 합성 데이터로 학습하면 실제 이미지도 일반화할 수 있다

1. 핵심 아키텍처

Depth Anything V2 = Vit 백본 인코더 + 멀티 스케일 특징 합성 + 깊이 밀도 복원 디코더 + 깊이 구간화(Depth Binning) 헤드

1. 핵심 파이프라인

2D RGB Image → Vit Feature Extration → Depth Binning Classification → Continuous Value Regression → Metric Depth Map Output

학습 단계는 “인공 데이터 준비 → Teacher 모델 학습 → Teacher로 실제 데이터에 라벨 생성 → Student 모델이 라벨링한 실제 데이터 학습 → 계속 반복”

- 반복에서 Student를 Teacher로 사용하기도 함

추론 단계는 “실제 이미지 입력 → Student 모델에 입력 → ViT Backbone으로 특징 추출 → 멀티 스케일 합성으로 정제 → 경량 디코더로 빠르게 처리 → 정밀 깊이 Map 출력”

1. 핵심 모듈

ViT Backbone (Vision Transformer)

- “Patch Embedding + Transformer Encoder Block + Multi-Scale Feature Output”으로 구성
- 이미지를 다중 스케일 특징으로 변환
- 구조화된 특징 벡터 추출
- 입력 영상의 전역적 시각 정보를 분석하여 원근감, 조명 변화, 오클루전 관계를 견고하게 인지
- 픽셀 단위가 아닌 패치 단위이므로 빠름

Adaptive Depth Binning 헤드

- 깊이 범위를 여러 개의 이산적인 칸(Bin)으로 나누어 물체가 어느 거리 범위에 속하는지 확률적으로 분류
- 미세 조정 회귀를 통해 연속적인 실제 미터 단위 거리 값으로 최종 보정

Multi Scale Feature Fusion

- 여러 깊이의 특징들을 효과적으로 병합
- Coarse-to-Fine을 구현
- Skip Connection을 통한 정보 손실 최소화

Lightweight Decoder

- 융합된 특징을 고해상도 깊이 맵으로 빠르게 변환
- 업샘플링 수행

1. 실험 결과

데이터가 없는 와일드(In-the-wild) 환경에서 경계 보존 능력 대폭 향상

- 오차가 포함된 실제 센서 데이터 대신 기하학적으로 완벽한 정답을 가진 고품질 가상 합성 데이터와 대규모 정제 데이터 루프를 혼합 학습에 활용했기 때문
- 기존 SOTA는 ZoeDepth
- 제로샷을 말하는듯

상대 깊이 오류 3배 개선

소형 모델로 모바일 실시간 구동 수준의 속도 달성

즉 합성 데이터, 증류 학습의 결과가 매우 큼

1. 논문 의의

인공적으로 만든 고품질의 데이터가 오차가 있는 실제 데이터보다 나음

- 즉, 인공적으로 만든 고품질 데이터 > 실제 데이터

파운데이션 모델 제시

- 제로샷 깊이 추정
    - 학습 없이 새로운 환경에 바로 적용 가능
- 도메인 특화 모델 필요 없음

모바일 실시간 처리 가능

- 스마트폰 수준에서 고품질 깊이 추정 가능

1. 한계점

실시간 비디오 스트림에 적용할 경우, 프레임 간 깊이 값이 미세하게 떨리는 지터링(Jittering) 현상이 여전히 존재

- 비디어 프레임 간 일관성 문제가 있음

절대 깊이(Metric Depth) 정보가 부족함

- 카메라 파라미터나 별도 스케일 정보 필요함

투명/반사 물체 인식 실패

어두운 환경에서 깊이 추정이 어려움

오클루전 문제

1. 결론

단안 카메라 환경에서도 고가의 LiDAR 없이 정밀한 절대 거리 측정이 가능한 범용 비전 파운데이션 모델 구축

합성 데이터와 자기지도 학습의 조합으로 범용 파운데이션 모델 구현

단안 RGB만으로 현실 공간의 3차원 구조를 추정할 수 있게 만든 파운데이션 모델

1. 내 연구에서는?

SAM으로 분할한 객체들과 Depth Anything V2의 값을 결합하여 → 카메라로부터 사물까지의 실제 거리 및 바닥 기준 높이를 수치 데이터로 산출

- SAM과 Depth Anything V2 중 무엇을 먼저 적용시킬지 결정해야 함

1. 궁금점

두뇌 역할을 하는 AI나 행동 역할을 하는 AR 아바타 입장에서는 절대 거리와 상대 거리 중 어떤 것이 더 유리한가?

Depth Anything V2가 리턴한 거리 정보를 어떻게 보정하는 것이 필요한가?

# AffordanceLLM

1. 한 줄 정리

시각 언어 모델 (VLM)에 내재된 인간-사물 상호작용 상식을 활용하여
→ 자연어 명령에 맞는 사물의 기능적 행동 가능 부위(Affordance)를 2D 픽셀 좌표 상에 가리키는 모델

1. 기존 문제점

기존 어포던스 검출 모델은 미리 정의된 한정된 클래스 내에서만 작동했음

- 유연한 맥락적 자연어 지시에 대응하지 못함

1. 논문 아이디어

이미 대규모 모델에 사물의 어포던스 정보가 내재되어 있음. 그러니 VLM의 언어 지식을 시공간 좌표계에 정렬(Align)시켜 추론하자.
